# VI Sensitivity Analysis

## **Workflow** 
<Steps>
### Data preparation
   * Load Sentinel-2 spectral bands (Blue, Red, NIR, Red Edge)
   * Aggregat 10 m bands to 20 m resolution for consistency
   * Project datasets to EPSG:25832
### NDVI calculation
   * Compute NDVI = (NIR − Red) / (NIR + Red)
   * Crop NDVI to national park boundaries and export as GeoTIFF
### NDRE calculation
   * Compute NDRE = (NIR − Red Edge) / (NIR + Red Edge)
   * Crop NDRE to national park boundaries and export as GeoTIFF
### Histogram analysis
   * Convert rasters (NDVI, NDRE) to data frames
   * Generate histograms with density overlays and mean reference lines
### Sampling and Scatterplot
   * Label sampling points based on given classes (deciduous forest, coniferous forest, dead trees, no forest)
   * Extract NDVI and NDRE pixel values for sampling points
   * Merg samples with class labels
### Visualization
   * Visualize NDVI and NDRE maps
   * Add histograms and scatterplot to map layout
</Steps>
